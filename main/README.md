# Environment

The code needs:
numpy
matplotlib
scipy
pillow
joblib
torch
torchvision
ipython

Run scripts from `final_project/main`, from 1 to 9 in order, it will generate the results under Results folder. 

Some scripts need `./data/bruno_sparse/IMAGES_Vanhateren.npy`. Which I included here in the ./data/ folder/ 

Run Script 6 will automatically download MNIST data. 

For 7_run_sparsecoding_bruno.py, this code and relevant functions is courtesy from: https://github.com/yubeic/Sparse-Coding/, where I modified the code a little bit to allow it works in my work flows.
